  # Health Prediction App
  
  ## Overview
  
  This is a Python based Health Prediction Application that predicts possible diseases using laboratory health parameters. The application also generates AI bsed health remarks and stores patient records in a database.
  
  ## Features
  
  * Disease prediction using a Machine Learning model
  * AI-generated health remarks using Gemini API
  * Patient data storage using SQLite
  * CRUD operations
  * Input validation for user data
  * Simple Streamlit-based user interface
  
  ## Technologies Used
  
  * Python
  * Streamlit
  * Pandas
  * Scikit-learn
  * SQLite
  * Google Gemini API
  
  ## Dataset
  
  The model was trained using the provided laboratory dataset containing multiple disease categories and patient laboratory parameters.

  ## How to Run
  
  1. Install required packages
  
  ```bash
  pip install -r requirements.txt
  ```
  
  2. Add your Gemini API key in the environment configuration.
  
  3. Run the application
  
  ```bash
  streamlit run app.py
  ```
